# App Flow - MetalVale Core

## Core Journey
1. **Dashboard**: High-level overview of sales, production, and inventory.
2. **Orders (Pedidos)**: Management list and detail view for e-commerce and factory orders.
3. **Products (Produtos)**: Catalog management.
4. **Clients (Clientes)**: CRM and history.
5. **Inventory (Estoque)**: Warehouse tracking and alerts.
6. **Reports (Relat√≥rios)**: Data-driven insights.

## Interaction Patterns
- **Navigation**: Persistent sidebar with collapse capability.
- **Search**: Global search in header.
- **Actions**: Top-right placement in page headers.
- **Drill-down**: Cards and table rows are clickable to navigate to details.
