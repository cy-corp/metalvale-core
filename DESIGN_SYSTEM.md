# Design System - MetalVale Core

## Visual Language
A premium, industrial-modern aesthetic inspired by high-end manufacturing and Apple-like minimalism. Quiet, confident, and effortless.

## Color Tokens
- **Primary**: `#EC1C24` (Industrial Red) - Used for primary actions and brand emphasis.
- **Secondary**: `#231F20` (Charcoal) - Used for high-contrast background and text.
- **Surface**: `#FFFFFF` (White) - Main content background.
- **Surface-Muted**: `#F8F9FA` (Off-white) - Dashboard background.
- **Border**: `#F1F1F1` - Subtle dividers.
- **Success**: `#059669` (Emerald) - Calmer than standard Green.
- **Warning**: `#D97706` (Amber) - Professional status.
- **Destructive**: `#DC2626` (Bright Red) - Errors.

## Typography
- **Headings (Avant)**: `Outfit`, sans-serif. Used for titles, metrics, and navigation labels.
  - Tracking: `-0.02em` for large titles.
  - Case: Sentential or Uppercase (reserved for tiny UI labels).
- **Body (Josefin)**: `Josefin Sans`, sans-serif. Used for long-form text, descriptions, and data tables.
  - Line Height: `1.5` for readability.

## Elevation & Radii
- **Radii**:
  - `sm`: `4px`
  - `md`: `8px`
  - `lg`: `16px` (Default for Cards)
  - `full`: `9999px`
- **Shadows**:
  - `premium`: `0 8px 30px rgba(0, 0, 0, 0.04)`
  - `active`: `0 4px 10px rgba(236, 28, 36, 0.15)`

## Spacing
- Based on an 8px grid system.
- Standard gutter: `32px` (`p-8`).
- Component spacing: `16px` (`gap-4`).
